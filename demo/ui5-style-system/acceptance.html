How To Use

Review Flow

The intended order is simple: use the matrix page for state-level checks, the scenario pages for page-level checks, and the token page whenever you need to confirm whether a problem comes from a wrong variable or a wrong layout rule.

1. Check states 2. Check worklist composition 3. Check object page composition 4. Check computed tokens 5. Check history feedback panel
