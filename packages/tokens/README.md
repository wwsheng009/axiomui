# @axiomui/tokens

Design token styles for AxiomUI.

## Installation

```bash
npm install @axiomui/tokens
```

## Usage

Import the published stylesheet from your application entry:

```ts
import "@axiomui/tokens/styles.css";
```

This package is also used by `@axiomui/react` as the shared token source for
component styling.
