# Sprint 3 Epic

## Type

- Sprint Epic

## Goal

把 AxiomUI 从基础表单层推进到可支撑复杂筛选、消息反馈和自适应弹层的企业交互层，并把这些能力接回现有 worklist 场景。

## Target Deliverables

- MultiComboBox
- DateRangePicker
- TimePicker
- ResponsivePopover
- Menu
- MessagePopover
- FilterBar 高级筛选集成
- 高级筛选 worklist demo

## Planned Task Breakdown

- [ ] 01 MultiComboBox
- [ ] 02 DateRangePicker
- [ ] 03 TimePicker
- [ ] 04 ResponsivePopover
- [ ] 05 Menu
- [ ] 06 MessagePopover
- [ ] 07 FilterBar 高级筛选集成
- [ ] 08 高级筛选 worklist demo
- [ ] 09 状态与帮助文案统一
- [ ] 10 A11y 与回归测试

## Exit Criteria

- [ ] MultiComboBox、DateRangePicker、TimePicker、ResponsivePopover、Menu、MessagePopover 已导出
- [ ] Worklist 高级筛选 demo 跑通
- [ ] 新控件与 FilterBar、FormGrid 协作稳定
- [ ] 核心键盘与焦点路径已验证
- [ ] docs 可作为主要 walkthrough 入口

## Main Demo

- Advanced Filter Worklist
- MessagePopover demo

## Main Risks

- 复杂组合控件状态过多，容易出现焦点和关闭行为不一致
- ResponsivePopover 的桌面与小屏模式切换可能引入回归
- FilterBar 中混排复杂控件可能再次触发布局问题
