# Sprint 5 Epic

## Type

- Sprint Epic

## Goal

把 AxiomUI 从“能做企业工作台和对象页”推进到“能表达企业 KPI 与趋势”的阶段，交付第一批 microchart 和 KPI 组合层。

## Target Deliverables

- chart primitives 与图形 token 基础
- BulletMicroChart
- RadialMicroChart
- DeltaMicroChart
- HarveyBallMicroChart
- StackedBarMicroChart
- InteractiveDonutChart
- InteractiveLineChart
- KPI Card
- Chart Lab

## Planned Task Breakdown

- [ ] 01 Chart primitives 与 token 基础
- [ ] 02 BulletMicroChart
- [ ] 03 RadialMicroChart
- [ ] 04 Delta 与 HarveyBall
- [ ] 05 StackedBarMicroChart
- [ ] 06 InteractiveDonutChart
- [ ] 07 InteractiveLineChart
- [ ] 08 KPI Card 组合件
- [ ] 09 Chart Lab 与业务示例接入
- [ ] 10 A11y、性能与回归测试

## Exit Criteria

- [ ] 第一批 microchart 已导出
- [ ] KPI Card 可嵌入真实页面
- [ ] docs 有独立 Chart Lab
- [ ] 至少 3 个真实业务场景接入图形组件
- [ ] 图形组件具备基础 a11y 文案与关键路径测试

## Main Demo

- Chart Lab
- KPI cards
- Object page summary
- Table/list inline indicators

## Main Risks

- 图形 token 不统一，后续视觉容易漂移
- 交互图表过早做复杂，拖慢整体节奏
- 图形组件只有静态 demo，没有真实业务嵌入
