# Meeting Kit

这组文档用于把 AxiomUI 的路线图、backlog、执行板继续收口成可直接使用的会议模板。

## 文档目录

- [weekly-sync-template.md](./weekly-sync-template.md)
  周会模板，适合跟踪 `本周完成 / 阻塞 / 下周计划 / 风险变化`。
- [milestone-review-template.md](./milestone-review-template.md)
  里程碑验收模板，适合 Sprint 结束或阶段交付 review。
- [risk-retro-template.md](./risk-retro-template.md)
  风险复盘模板，适合处理延期、返工、依赖失效等问题。
- [decision-log-template.md](./decision-log-template.md)
  架构和产品决策记录模板，避免口头结论丢失。

## 建议使用方式

1. 每周固定使用 [weekly-sync-template.md](./weekly-sync-template.md)。
2. 每个 Sprint 结束使用 [milestone-review-template.md](./milestone-review-template.md)。
3. 当出现结构性返工、依赖阻塞或重大偏差时，补一份 [risk-retro-template.md](./risk-retro-template.md)。
4. 当出现 API 冻结、主题命名、组件边界等关键决策时，补一份 [decision-log-template.md](./decision-log-template.md)。
